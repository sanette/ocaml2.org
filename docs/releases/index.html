Releases



The latest page points to the most recent release of the
OCaml compiler distribution. Below is a list of the recent releases.


See also the install page for instructions on
installing OCaml by other means, such as the OPAM package manager and
platform specific package managers.


  • OCaml 4.10.0, released Feb 21, 2020.
  • OCaml 4.09.1, released Mar 18, 2020.
  • OCaml 4.09.0, released Sep 18, 2019.
  • OCaml 4.08.1, released Aug 5, 2019.
  • OCaml 4.08.0, released Jun 14, 2019.
  • OCaml 4.07.1, released Oct 4, 2018.
  • OCaml 4.07.0, released Jul 10, 2018.
  • OCaml 4.06.1, released Feb 16, 2018.
  • OCaml 4.06.0, released Nov 3, 2017.
  • OCaml 4.05.0, released July 13, 2017.
  • OCaml 4.04.2, released Jun 23, 2017.
  • OCaml 4.04.1, released Apr 14, 2017.
  • OCaml 4.04.0, released Nov 4, 2016.
  • OCaml 4.03.0, released Apr 25, 2016.
  • OCaml 4.02.3, released Jul 27, 2015.
  • OCaml 4.02.2, released Jun 17, 2015.
  • OCaml 4.02.1, released Oct 14, 2014.
  • OCaml 4.02.0, released Aug 29, 2014.
     (4.02.0 suffers from one known bug that noticeably increases compilation time, you should use 4.02.1 or later versions instead.)
  • OCaml 4.01.0, released Sep 12, 2013.
  • OCaml 4.00.1, released Oct 5, 2012.
  • OCaml 3.12.1, released July 4, 2011.
  • Earlier releases are available
 here.



You may also want to check the development version of
OCaml.
